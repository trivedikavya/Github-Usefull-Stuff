**File Path:** `Templates/Standards/.editorconfig`
**What it does:** If a user drops this in their root folder, VS Code will automatically fix their indentation (spaces vs tabs) to match the team standard.

```ini
# http://editorconfig.org
root = true

[*]
charset = utf-8
indent_style = space
indent_size = 2
end_of_line = lf
insert_final_newline = true
trim_trailing_whitespace = true

[*.md]
trim_trailing_whitespace = false
```
